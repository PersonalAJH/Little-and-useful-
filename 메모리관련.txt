C++ 에서 스택메모리와 힙메모리 

스택메모리 

사용방법 
- 함수 내에서 선언되는 로컬 변수들은 모두 스택에 할당된다.
- 함수의 매개변수와 반환 주소도 스택에 저장된다.()

특징
- 메모리 할당과 해제가 자동으로 이루어진다.(컴파일러의 능력)
- 스택 오버플로우 위험성이 있으나, 일반적인 상황에서도 가끔 나타난다(?)
- 접근 속도가 빠르고 메모리 관리가 간단하다.

예시
void function(int test){
    int local_variable; // 스택에 할당된다.
}




힙 메모리

사용방법
- new 키워드를 사용하여 동적으로 메모리에 할당한다. 
- 할당된 메모리는 delete 키워드로 명시적으로 제거해야한다. 

특징
- 프로그래머가 메모리 할당과 해제를 직접합니다. 
- 메모리누수(memory leak)과 같은 문제가 발생할 수 있으므로 조심해야한다. 
- 스택에 비해 큰데이터를 저장하거나 생명주기가 긴 데이터를 관리하는데 효과적임


얘시
int* dynamicVar = new int(10); // 힙에 할당
delete dynamicVar; // 할당 해제
