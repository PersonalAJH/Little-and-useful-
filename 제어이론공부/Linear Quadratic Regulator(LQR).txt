선형 2차 조절기(Linear Quadratic Regulator, LQR)는 선형 시스템의 상태를 원하는 목표 상태로 안내하기 위해 설계된 최적 제어 알고리즘입니다. LQR은 제어 이론에서 가장 기본적이고 널리 사용되는 방법 중 하나로, 시스템의 성능을 최적화하는 제어 입력을 계산하는 데 사용됩니다.

이름 그대로 Linear(선형) Quadratic(에러등 줄이기 위한 내용을 제곱) Regulator 줄여주는 제어기

기본원리
1. LQR 의 설계의 목적은 주어진 선형 시스템의 성능을 최적화 하기 위해 제어 입력을 계산하는것. 이를 위해 LQR 은 제어 입력의 크기와 시스템 상태의 편차를 동시에 최소화 하는 2차 비용함수를 최소화한다. 

x'(t) = A * x(t) + B * u(t)   -> x' 은 미분 x(t) 는 상태변수, u(t) 는 입력변수 

비용함수는 .... 이걸로 어떻게 쓰지.. 
j = integral(무한대~0까지) (x^t * Q * x + u^t * R * u)dt 

여기서 Q 는 상태벡터의 각 요소에 대한 가중치를 정의하는 대칭이고 양의 준정부호 행렬이다. 
R 은 제어 입력의 각 요소에 대한 가중치를 정의하는 대칭이고 양의 정 부호 행렬이다. 


LQR 의 장점 
효율설 : LQR은 선형 시스템에 대해 잘 정의되고 효율적으로 계산할 수 있는 최적 제어 법칙을 제공함
간단함 : LQR 설계 절차는 이해하기 쉽고 구현하기 간단함
RObustness : 적잘한 가중치 선택을 통해 외란과 노이즈에 대해서 효과가 미비함(?) 강인함
