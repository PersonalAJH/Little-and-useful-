라그랑주 승주법(Lagrange Multipliers)은 주어진 제약 조건 하에서 함수의 극값(최대값 또는 최소값)을 찾는 수학적 방법. 이 방법은 특히 제약 조건이 있는 최적화 문제에서 매우 유용하며 다양한 분야에서 활용 된다. 

f(x,y) = x^2 + y^2 의 최소값을 구하며 조건이 x+y+1 = 0 이라고 했을때 

1. 라그랑주 함수 구성 
    L(x,y,l) = x^2 + y^2 + l * (x + y + 1)

2. 편미분 
    dL/dx = 2x + l = 0
    dL/dy = 2y + l = 0
    dL/dl = x + y - 1 = 0


3. 편미분 방정식 풀기 
    x=y=1/2 l = -1 

4. f(x,y)는 x=y=1/2 에서 최솟값을 가진다.
